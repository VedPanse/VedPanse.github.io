What have I been upto?

Detecting Fake News Using Machine Learning

Currently, I am working on detecting fake news on social media using Machine Learning. Solving this problem intrigued me because it led to chaos during the COVID-19, especially in India, as unverified home remedies for tackling the virus surfaced through social media.

I have written an article for the newspaper to make the general audience aware about almost every basics of this issue: from its impacts to how it is being handled. I have uploaded my version of solving this issue on GitHub. This program uses primitive machine learning methods and a CSV file to train as well as predict if a news is fake. My solution is not 100% accurate, but it is still a start.
