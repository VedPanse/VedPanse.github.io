My Projects



Amazon Price Tracker


A program that takes in the budget of the user, the targeted item, and sends an email to the provided email address when the price of the item on Amazon falls below the budget. Built on Python.

ISS Notifier


A program that notifies you when the ISS is in close proximity with your position. Taking the current latitude and longitude of the user via input, it gets the position of the ISS through an API and emails the user on the email address provided beforehand when the ISS is overhead.

Workout Tracker


A program, that keeps in track your daily workout record and notifies you about your weekly progress. Written in Python, this program uses an interface from API provided by Sheety.
