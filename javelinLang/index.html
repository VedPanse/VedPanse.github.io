Javelin Language

Written By Ved Panse Last Modified: 30 June 2021
javelinLang is an open-source Domain Specific Language (DSL), programmed principally for generating Graphical User Interface (GUI). I felt a need to develop it to avoid typing long lines of code to achieve the desired GUI output. The language, though not completed yet, helps the programmer to save time.



Benefits of javelinLang

Easily Understandable

javelinLang has an easy-to-understand syntax, making it easy for even budding Python programmers to understand programming. This benefit easily overcomes its disadvantage: a rigid and strict structure.

Fast

javelinLang was built using Python. Thus, it makes it have the same processing and execution speed as Python has. This quality provides for a good user experience (UX) too.

Incredible Growth

There has been incredible growth in the number of people who choose to learn Python. javelinLang, being built using Python, has a high chance of seeing growth in its features due to the wide availability of open source contributors.
