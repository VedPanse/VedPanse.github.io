Basic Syntax

Basic Syntax

Written By Ved Panse Last Modified: 30 June 2021
In javelinLang, special symbols (!, $, |||, etc.) distinguish between different data structures and data types. These symbols indicate what comes next in line.



Defining Variables

javelinLang has a specific way of defining variables. A "$" sign indicates that in that line, we will be defining a variable.


Suppose we want to define a variable 'name' with 'Clay' as its value. 

$name = "Clay"


javelinLang, unfortunately, has a rigid structure. But, on the bright side, this rigid structure helps to maintain a clean code. Whatsoever, if not carefully typed, the program might encounter errors. Spaces before and after the '=' are necessary. The following code may give errors.

$name="Clay"


$name ="Clay"


$name= "Clay"


Space after the "$" sign might also encounter errors.

$ name = "Clay"


Calling Functions

javelinLang contains predefined functions like println, create_window, etcetera. Typing a "!" in the beginning will indicate that the statement/expression written ahead deals with a function.



Print Statements

The print statement in javelinLang is quite easy: !println

!println -> "Hello, World"
